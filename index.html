Spartan CSS (WIP!)

A minimalist CSS framework for modern browsers. Inspired by Skeleton CSS. Download a minified Spartan CSS - size: 999 bytes minified.

Simply download the minified Spartan CSS and include it like so: 

<link type="text/css" rel="stylesheet" href="css/spartan.min.css"/>

This is my own framework that I developed because I like Skeleton CSS's basic ideas, but ultimately the font and excessive padding are things I do not like. So, I made my own smaller CSS mini-framework for my own projects. Hopefully others will like it and potentially use it as well.
